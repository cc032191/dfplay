    # 一.Tiktok闪退如何解决？

    需要使用下载tiktok网页中提供的美区账号在苹果商店中任意下载一款免费软件，即可解决。

    解决方法如下：打开你刚下载tiktok的网页，找到你之前安装软件对应的苹果ID，登录App Store下载任意一款软件即可解决闪退问题（tiktok除外）。

    # 二.Tiktok黑屏如何解决？

    我们要了解为什么会黑屏，因为tiktok是不允许国内访问的，出现黑屏就是因为检测到国内信息
    教程中的配置小火箭安装信任证书，就是给手机模拟国外环境。
    如果出现黑屏按以下方法排查：

    # 1.检查tiktok版本

    首先我们先检查下tiktok版本是不是21.1.0，如果不是请卸载
    打开tiktok-个人资料-右上角三个横杠-拉到最下面有版本号，如果版本号不是V21.1.0，则表示不是从教程里提供的网址下载的，一定不要从商店下载，要不然是黑屏！

    如果版本是V21.1.0，那继续检查下面的步骤是否正确。

    # 2.使用国内、香港IP或者印度IP(本店免费节点则不需要)

    tiktok在一些国家和地区是禁用的，使用这些地区的IP访问会黑屏， 要选择日本、韩国、美国、英国、新加坡、俄罗斯等节点。

    # 3.证书没有添加信任

    打开设置-通用-关于本机-证书信任设置检查是否勾选

    # 4.证书操作错误

    • 检查默认配置文件是否是苹果免拔卡配置文件

    • 证书信任设置中的时间与小火箭配置中的时间必须一致

    解决方案：重新配置一遍小火箭

    # 5.没有导入节点或者节点选择错误

    # 6.小火箭全局路由没有修改成代理

    # 7.手机没有插卡或者Ipad

    如果是没插卡的手机或者ipad，需要打开手机设置-通用-语言与地区，把【地区】设置为美国，语言不需要修改默认中文，只需要改地区就可以。

    # 8.提前打开过tikotk

    没配置好环境，没连接国外节点，提前打开过Tiktok，被tktok检测到异常，需要删除tiktok，重新下载

    如果以上步骤都确定没有问题，就把tiktok卸载掉，重新安装一遍即可恢复正常。

    # 三.提示更新Tiktok

    解决方法如下：
    如果提示请更新tiktok到最新版本，请从后台关掉tiktok，然后重新打开，先注册登录之后，就不会在提示了
